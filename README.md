# Pâtisserie Moriguchi（地元ケーキ屋さんのECサイト）
## 概要
最低限の機能（カート、決済方法の選択、マイページ、販売ステータス、退会、商品管理、配送先管理等）を実装しました！

## 作成した背景
ネットでは販売しておらず知名度が低いお店を中心に地元以外の方にも知ってもらいたい、そして地元の活性化をしたいという目的と
ECサイトにチャレンジしたいという気持ちがあったため、作成しました。
